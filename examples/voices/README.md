# Voice clone reference clips

Bundled demo reference for clone mode. Copied to `packages/voice-runtime/voices/ref.wav` on first launch.

| File | Purpose |
|------|---------|
| `ref.wav` | ~10s clean reference clip (default voice) |
| `ref.txt` | Transcript for ICL mode (`xvec_only` off) |

Upload your own clips via **Settings → Voice** or drop WAVs in `packages/voice-runtime/voices/`.
